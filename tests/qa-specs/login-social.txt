Feature: Social login options

Users can authenticate using social login providers.

Social login available:
- User opens the login page.
- Social login section is displayed below form.
- Social login buttons are visible.
- User can choose social provider.

Social login in progress:
- User opens the login page.
- User clicks social login button.
- System disables email/password form.
- Social login loading state is active.
- User cannot submit email/password form.

Social login success:
- User initiates social login.
- Authentication succeeds with provider.
- System stores access and refresh tokens.
- User is authenticated.
- Login modal closes.
- User is redirected if redirect path exists.

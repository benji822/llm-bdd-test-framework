Feature: Login security controls

Handles account protection and security restrictions.

Account lockout:
- User account has exceeded maximum login attempts.
- User enters valid email and password.
- User clicks submit button.
- System shows error "Your account has been locked".
- Error message includes support link.
- Support link opens in new tab.

Location blocked:
- User attempts login from restricted location.
- User enters valid credentials.
- User clicks submit button.
- System immediately redirects to restricted page.
- No authentication occurs.

Self-excluded account:
- User account has active self-exclusion timeout.
- User enters valid credentials.
- User clicks submit button.
- System prevents login.
- Timeout modal displays exclusion details.
- Modal shows start time and end time.
